# KDR

### Khys Dai Roguelike

---

KDR is a simple roguelike rendering/input/audio framework.


## Externals Used


Rendering:

* GLEW 

* GLFW

* FreeType

* FreeType-GL


Input:

* GLFW


Audio:

* IrrKlang


Gameplay:

* FastNoise

---

# Features So Far


* Window creation

* Input

* Textures

* Vertex and Fragment shaders

* Tile Rendering

* Batch Rendering

* Text Rendering

* Rendering transformations
